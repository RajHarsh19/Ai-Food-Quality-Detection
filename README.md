# AI Food Quality Detection

AI-powered food freshness detection system using:

- React
- FastAPI
- TensorFlow
- OpenCV
- Deep Learning

## Features

- Food freshness prediction
- Confidence score
- AI image analysis
- Responsive modern UI

## Tech Stack

Frontend:
- React
- Tailwind CSS

Backend:
- FastAPI
- TensorFlow
- OpenCV

## Run Frontend

cd frontend
npm install
npm start

## Run Backend

cd backend

pip install -r requirements.txt

uvicorn app.main:app --reload
